# Scooper Heroes Website

Static website for Scooper Heroes dog waste removal services.

## Main Files

- `index.html` - homepage
- `services.html` - services page
- `service-area.html` - service area page
- `about.html` - about page
- `faq.html` - FAQ page
- `contact.html` - contact page
- `privacy-policy.html` - privacy policy
- `styles.css` - site styling
- `script.js` - mobile navigation and small interactions
- `assets/` - images, video, and logo files

## Publishing

Upload the contents of this folder to the website host's `public_html` folder, or connect this folder/repository to Hostinger Git deployment.

The live homepage file must be directly at:

`public_html/index.html`
